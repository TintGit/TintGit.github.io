高效工具

发表于 2018-04-12

Xcode

  • shift + command + o 弹出快速查找文件窗口
  • shift + command + j 定位到文件所在目录
  • command + t/w 新建/关闭一个Tab
  • command + alt + 左右 收缩代码
  • control + i 缩进
  • command + n新建文件 command+shift+n 新建工程
  • command + alt + [/] 上移/下移

其他

  • 编辑器 多行编辑 shift + command + L
  • shift + command + 3全图截屏
  • shift + command + 4自选区域截图
  • command + F3 快捷查看桌面

iTerm2 - 终端利器

快捷键

命令 说明
command + d 垂直分屏
command + shift + d 水平分屏
command + ; 查看历史命令
ctrl + u 清除当前行
command + shift + h 查看剪贴板历史
ctrl + u 清除当前行
ctrl + a 到行首
ctrl + e 到行尾
ctrl + p 上一条命令
ctrl + r 搜索命令历史
command + r 清屏

Hexo - 快速、简洁且高效的博客框架

命令

  • hexo generate -> 生成静态文件
  • hexo deploy -> 部署网站
  • hexo server -> 启动服务器
  • hexo server -p xxxx 重设端口
  • ctrl + c 取消

通用命令

lsof -i:8080 -> 查看8080端口号占用
kill -9 xxxx -> kill进程
